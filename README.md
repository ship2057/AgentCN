# AgentCN – 你的原生 AI 助手

AgentCN 是基于 llama.cpp + PySide6 的 Windows 原生 AI 代理，支持：

- 逐字流式对话
- 项目级上下文隔离
- 原生插件系统（搜索、RPA、屏幕感知）
- 技能导出 / 调度
- GUI + CLI 双模式

## 快速开始

```bash
pip install -r requirements.txt
python agenthub_main.py --gui
